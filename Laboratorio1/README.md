# Laboratorio 1

Este laboratorio tiene como objetivo evaluar:

- El manejo basico de Zilly (visto en la guia computando con Zilly).
- El manejo basico de C++.
- Manejo de arreglos en C++ (visto en el apartado de vectores/arreglos)
- Manejo de recursion (visto en la primera clase en la F-machine).


## Instrucciones para el archivo `lab1.z`

Completar las funciones descritas en el archivo. Estas
deben realizarse en el lenguaje Zilly.

## Instrucciones para el archivo `lab1.cpp`

Completar las funciones descritas en el archivo. Estas
deben realizarse en el lenguaje C++.

## Instrucciones Adicionales

- El programa en c++ debe poder compilarse usando el comando
`g++ lab1.cpp -o lab1.sh` y ejecutarse con `./lab1.sh`. De lo contrario no se corregira

- El programa en zilly debe poder interpretarse sin errores. De lo contrario no se corregira.

- No se debe eliminar ninguna de las funciones ya definidas en los archivos. De lo contrario no se corregira.

- La funcion `show` del archivo `lab1.cpp` ya esta implementada y no se debe modificar.

- El laboratorio sera evaluado en base a 100. Cada archivo vale 50 puntos. Y cada pregunta vale un numero igual de puntos.
